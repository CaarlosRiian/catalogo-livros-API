# catalogo-livros-API

<hr>

1. Instale as Dependencias:
```sh
pip install -r requirements.txt
```
2. Crie o Ambiente Virtual:
```sh
python -m venv venv
```
3. Ativar Ambiente Virtual:
```sh
[Windows] .\venv\Scripts\activate
[Linux/Mac] source venv/bin/activate
```
